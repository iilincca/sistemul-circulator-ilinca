🫀 Sistemul Circulator

Site educațional interactiv despre sistemul circulator uman, disponibil în română și engleză.

🔗 Link site
https://iilincca.github.io/sistemul-circulator-ilinca

 📚 Funcționalități
- Conținut educațional structurat pe secțiuni (inimă, artere, vene, capilare, sânge, grupe de sânge, Rh, Rh-nul, circulație)
- Simulator interactiv de grupe sanguine
- Test grilă cu verificare și feedback instant
- Video animație circulația sângelui
- Comutare RO/EN completă
- Dark Mode cu persistență
- Funcționează offline (localStorage)

🛠️ Tehnologii
HTML5, CSS3, JavaScript, localStorage, GitHub Pages, Git

 📁 Structura fișierelor
- `index.html` — structura paginii
- `style.css` — stilizare și responsive design
- `script.js` — funcționalități interactive
